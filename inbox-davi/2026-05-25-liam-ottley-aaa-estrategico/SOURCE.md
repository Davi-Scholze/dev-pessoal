### FRAMEWORK 2: AIOS — AI Operating System
**O que é:** sistema operacional de IA para o negócio do cliente.
Não é um chatbot. É a operação inteira rodando com IA.

**Definição de Liam:**
"An AI Operating System is the complete infrastructure that allows
a business to operate intelligently — agents handling tasks,
memory preserving context, and workflows automating processes."

**Componentes de um AIOS:**

AI Agents         → executam tarefas específicas
Memory Layer      → preservam contexto do negócio
Workflow Engine   → automatizam processos operacionais
Integration Hub   → conectam ferramentas existentes
Dashboard         → visibilidade e controle
Copilot Interface → interface para o dono e equipe


**Como vender AIOS para SMBs:**
- Preço: $5.000 setup + mensalidade ($500-2.000/mês)
- Prazo: 7-14 dias para montar o sistema base
- Ferramenta principal: Claude Code (para desenvolvimento rápido)
- Pitch: "você vai ter um sistema operacional completo rodando IA
  na sua empresa em 7 dias"

como "funcionários digitais".

**Conceito central:**
Agentes = trabalhadores digitais que executam tarefas reais,
tomam decisões e operam 24/7 sem supervisão constante.

**Tipos de agentes que Liam ensina:**
1. Lead Gen Agent (qualifica e pesquisa leads)
2. Outreach Agent (personaliza e envia mensagens)
3. Support Agent (responde clientes automaticamente)
4. Research Agent (pesquisa mercado, concorrentes, dados)
5. Operations Agent (atualiza CRM, gera relatórios)
6. Voice Agent (atende ligações, qualifica por telefone)

**Ferramentas ensinadas:**
- n8n (workflows visuais — principal ferramenta)
- Make (Integromat)
- Relevance AI
- Claude / GPT-4o como LLM base

## 10. O VOCABULÁRIO DE LIAM OTTLEY (glossário para o KOD.AI)

**AAA (AI Automation Agency):** modelo de negócio criado por Liam para
vender automações de IA. Consiste em agência especializada + nicho +
setup fee + mensalidade.

**AIOS (AI Operating System):** sistema operacional de IA que gerencia
a operação completa de uma empresa. Evolução do AAA, produto premium
de 2025/2026.

**Digital Workers:** agentes de IA que executam tarefas como funcionários
reais — não apenas respondem perguntas.

**Setup Fee:** cobrar uma taxa única de instalação/configuração do sistema
(tipicamente $2.000-$10.000).

**Retainer:** mensalidade recorrente após o setup ($500-$3.000/mês).

**Build-in-public:** estratégia de Liam de mostrar o que está construindo
e aprendendo ao vivo no YouTube — cria conteúdo enquanto aprende.

**n8n:** plataforma de workflow principal que Liam usa e ensina.
"The backbone of every AI automation I build."

**Claude Code AIOS:** usar Claude Code especificamente para construir
o AIOS do cliente de forma rápida (7 dias).

**AI Agent Workforce:** time completo de agentes autônomos funcionando
em paralelo para cobrir diferentes funções de uma empresa.
