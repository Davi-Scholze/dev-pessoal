# Design Guide — Davi Scholze Dev

> Identidade visual pessoal/profissional.
> Usar como referência em qualquer entrega visual (slides, carrosséis, dashboards, UIs pessoais).

---

## Estilo geral

**Minimalista escuro.** Dark mode com tipografia limpa e poucos elementos visuais.
Sem excessos decorativos. Cada elemento tem função.

---

## Paleta de cores

| Token | Hex | Uso |
|---|---|---|
| `--color-bg` | `#0f0f23` | Fundo principal |
| `--color-surface` | `#1a1a2e` | Cards, painéis |
| `--color-surface-raised` | `#16213e` | Elementos elevados |
| `--color-accent` | `#4ecca3` | Destaque, links, foco |
| `--color-accent-alt` | `#e94560` | CTAs, alertas, urgência |
| `--color-text` | `#e0e0e0` | Texto principal |
| `--color-text-muted` | `#8892a4` | Texto secundário |
| `--color-border` | `#2a2a4a` | Bordas, divisores |

## Tom visual

- Sem gradientes excessivos
- Bordas finas (1px) em `--color-border`
- Raios de borda: 6–8px (sutis, não arredondados demais)
- Ícones: Lucide ou Phosphor (linha fina)
- Espaçamento generoso — não comprimir elementos
