# Prompts Reutilizáveis
> Armazene aqui prompts que funcionam bem e que você quer reusar entre projetos.

## Como usar

Copie o prompt e cole no início de uma sessão, ou adapte conforme necessário.

## Prompt de início de sessão (padrão)

```
Leia _negocio/MAPA.md e CLAUDE.md antes de começar.
Projeto atual: [NOME_DO_PROJETO]
Objetivo desta sessão: [OBJETIVO]
Contexto adicional: [OPCIONAL]
```

## Prompt de retomada de sessão

```
Estou retomando o projeto [NOME].
Leia _negocio/contextos/CONTEXTO_GERAL.md para entender o estado atual.
Última sessão: [DATA]
O que foi feito: [RESUMO]
Continuar de: [PONTO_DE_PARADA]
```

## Prompt de revisão de código

```
Revise o código em [ARQUIVO] com foco em:
- Segurança e vulnerabilidades
- Performance
- Qualidade e manutenibilidade
Use a skill code-review.
```
