# contextos/bruto/

Arquivos aqui são contexto **bruto** — chegaram em sessão como mensagem, transcrição, print ou ideia solta.

**Regras:**
- Não reescrever, não resumir, não estruturar — guardar como recebido
- Não usar como base para código diretamente
- Para virar código/spec, precisa ser promovido para `../fluxos/` com OK explícito do Davi

**Convenção de nome:** `YYYY-MM-DD_descricao-curta.md`
