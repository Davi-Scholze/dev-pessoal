# O QUE É UM "SITE TOP DE LINHA" INTERNACIONAL

5 pilares:
1. **Estética moderna** — tipografia grande, espaço em branco generoso, paleta refinada, layout assimétrico
2. **Interatividade** — micro-animações, hover states, scroll narrativo, cursor customizado, transições entre páginas
3. **Performance** — lazy-loading, WebP, compressão agressiva, CDN
4. **Responsividade** — adaptações reais em desktop, tablet e mobile (não só "encolher")
5. **Mídia rica otimizada** — vídeos em background, animações Lottie, WebGL/Three.js, tudo leve
